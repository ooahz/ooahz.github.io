一些Git的基础使用技巧

1. 前言

使用之前先来了解一下什么是GIT。
百度搜索词条“Git” ，或者点GIT查看。

2. 下载与安装

百度搜索Git，找到Git官网，或者点Git官网快速进入。
进入官网后，找到“Downloads” 选择合适的版本进行下载，或者直接点击右方的 “Download 2.26.2 for Windows“ 快速下载。
下载完成后像正常软件一样安装即可。

3. SSH秘钥连接

安装完成Git后，在电脑桌面右键，选择“Git Bash Here”。在弹出的操作窗口输入ssh-keygen，敲回车确定，然后看到提示秘钥以保存在xxx路径下（如果没有出现提示就多敲几次回车）。
在提示的路径下找到“id_rsa”文件，复制文件内容。
登陆你的Github帐号，进入“settings”，点击”SSH and GPG keys”，输入刚才复制的密钥命名保存。



3.Git的使用

  • 下载
    在任意路径下新建一个文件夹，作为Git操作的仓库。在此文件夹内右键，选择“Git Bash Here”。
    在弹出的操作界面输入git init 回车确定。然后可以看到该路径下生成”.git” 文件夹。由于该文件夹是隐藏的，请先确认你的电脑是否显示隐藏文件（百度搜索：电脑如何显示隐藏文件）。
    准备工作完成后，开始下载。
    在Github中找到需要操作的仓库，点击”Clone or down”，复制链接。
    在Git操作界面输入git remote add origin 刚刚复制的网址 ，回车。然后输入 git pull origin master，回车。耐心等待下载完成（该过程可能需与输入你的Github用户名及密码，按照提示输入即可）。

  • 上传
    在之前创建的Git操作文件夹右键，选择“Git Bash Here”。
    在弹出的操作界面输入git add 需要上传的文件名回车确定（如果需要上传该路径下全部文件，就输入git add . 即可）。
    输入git commit -m "文件备注（必填）"回车确定
    输入git push origin master回车确定，耐心等待上传完成（该过程可能需与输入你的Github用户名及密码，按照提示输入即可）。
    进入Github仓库，查看文件是否上传完成。
